AYLIEN Text Analysis API Demo

The Text Analysis API plugin by AYLIEN extracts structured information and insights from text. It performs Summarization, Hashtag Suggestion and Sentiment Analysis for any piece of text.

Important
inside AppBuilder open Plugins, AYLIEN-Text-Analysis-API, plugin.xml and do as this comment suggests (twice): <!-- replace with your own API key -->

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS - App package.
